# Metodo Founder

Workflow interativo que conduz founders na construcao de marca pessoal estrategica, transformando trajetoria em autoridade, autoridade em narrativa, e narrativa em crescimento empresarial real.

Baseado no metodo **Founder-Led Growth** criado por [Allan Barros](https://br.linkedin.com/in/allan-barros-acelerai) ([@allan__barros](https://www.instagram.com/allan__barros/)) — mentor G4.

## O que faz

Um comando (`/metodo-founder`) que conduz o founder por 9 estagios:

| Estagio | O que faz | Output |
|---------|-----------|--------|
| **Diagnostico** | Mapeia estagio atual (1-5) e pontua 8 dimensoes | Radar de autoridade com gaps e forcas |
| **Extracao da Essencia** | Identifica trajetoria, provas e tese emergente | Fundacao de Autoridade documentada |
| **Framework FOUNDER** | 7 modulos (Foundation → Relevance) | Engenharia completa da marca pessoal |
| **Framework LED** | Lideranca intelectual + educacao + direcao | Agenda de lideranca de narrativa |
| **Framework GROWTH** | Conecta autoridade com resultado empresarial | Plano de gravidade + oportunidades |
| **Posicionamento** | Consolida tudo em posicionamento acionavel | Manifesto, bio, pilares, tom de voz |
| **Manual** | Organiza tudo em documento consultavel | Founder-Led Growth Playbook (.md) |
| **Agenda 90 dias** | Transforma estrategia em acao concreta | Rotina semanal + roadmap 30-60-90 |

## 3 Frameworks

- **F.O.U.N.D.E.R** — Foundation, Ownership, Utility, Narrative, Distribution, Ecosystem, Relevance
- **L.E.D** — Leadership, Education, Direction
- **G.R.O.W.T.H** — Gravity, Reach, Opportunity, Wealth, Trust, Horizon

## Instalacao

Cole este prompt no G4 OS:

```
Instale o workflow "metodo-founder" de https://github.com/Gestao-Quatro-Ponto-Zero/g4os-skills — clone o repo, copie workflows/metodo-founder/ para o diretorio de workflows do meu G4 OS, e confirme que esta funcionando. Depois me apresente o que o workflow faz e como usar.
```

### Alternativa manual

```bash
git clone https://github.com/Gestao-Quatro-Ponto-Zero/g4os-skills.git
cp -r g4os-skills/workflows/metodo-founder ~/.g4os/workspaces/my-workspace/workflows/
```

## Estrutura

```
metodo-founder/
├── WORKFLOW.md              <- Instrucoes do workflow (o "cerebro")
├── README.md                <- Este arquivo
└── knowledge/               <- Referencias e guias
    ├── frameworks.md              <- Os 3 frameworks completos (FOUNDER, LED, GROWTH)
    ├── diagnostico-criterios.md   <- 5 estagios + scoring 8 dimensoes
    ├── posicionamento-templates.md <- Templates para posicionamento final
    └── agenda-90-dias.md          <- Rotina semanal + roadmap 30-60-90
```

## Requisitos

- **G4 OS** (qualquer versao com suporte a workflows)
- Sem dependencias externas

## Exemplos de uso

```
/metodo-founder
> "Quero construir minha marca pessoal como founder"
→ Inicia o processo completo de 9 estagios

/metodo-founder
> "Preciso me posicionar como referencia no meu mercado"
→ Inicia com foco em autoridade e posicionamento
```
