# Guia de Scoring de Canais

Referência para ajudar o usuário a pontuar canais quando não sabe o score.

## Escala de Validação (0-10)

| Score | Significado | Indicadores |
|-------|-------------|-------------|
| 0-1 | Nunca testou | Sem gasto, sem dados, só ideia |
| 2-3 | Teste inicial | Gastou pouco, 1-2 campanhas, dados insuficientes |
| 4-5 | Em validação | 3-6 meses investindo, dados emergindo mas inconsistentes |
| 6-7 | Validado | Dados de 6+ meses, consegue projetar com razoável confiança |
| 8-9 | Muito validado | 1+ ano de dados, ROI previsível, consegue escalar com confiança |
| 10 | Referência | Histórico longo, benchmark claro, altamente previsível |

## Escala de Potencial de Receita (0-10)

| Score | Significado | Indicadores |
|-------|-------------|-------------|
| 0-1 | Sem potencial | Audiência irrelevante para o negócio |
| 2-3 | Nicho | Funciona para um segmento pequeno, teto baixo |
| 4-5 | Moderado | Pode gerar receita consistente mas limitada |
| 6-7 | Alto | Pode ser um dos top 5 canais de receita |
| 8-9 | Muito alto | Pode ser top 3, alta escalabilidade |
| 10 | Ilimitado | Canal principal, sem teto visível de escala |

## Perguntas para Calibrar o Score

Se o usuário não souber pontuar, usar estas perguntas:

**Para Validação:**
- "Há quanto tempo investe nesse canal?" (< 3 meses = 2-4, 3-12 meses = 4-7, > 1 ano = 7-9)
- "Consegue prever quanto vai gerar no mês que vem?" (Não = 1-4, Mais ou menos = 5-7, Sim = 8-10)
- "Tem LTV:CAC calculado?" (Não = -2 pontos, Sim = +2 pontos)

**Para Potencial:**
- "Seu ICP está nesse canal?" (Não = 1-3, Parcialmente = 4-6, Sim = 7-9)
- "Se dobrasse o investimento, acha que o resultado dobraria?" (Não = 1-4, Talvez = 5-7, Sim = 8-10)
- "Concorrentes investem forte nesse canal?" (Não = sinal de nicho ou oportunidade, Sim = validação de potencial)
